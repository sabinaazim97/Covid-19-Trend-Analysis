The novel Coronavirus (COVID-19) is a highly infectious disease that has affected virtually every nation in the world. News first broke about this virus that started in Wuhan, China at the end of 2019. This disease continuous to appear throughout the world through 2021, with new variants emerging regularly. However, the introduction of COVID-19 vaccines have started to help slow that disease transmission.



Purpose of Project:

The purpose of this project is to work with Amazon Web Services to learn more about different services offered. This project will visualize which countries are leading in both disease transmission and vaccine administration. This project was created to observe trends that identifies how nations across the world are handling the COVID-19 pandemic. Some of the factors that this project will address include the number of testing, deaths, and vaccine administration.

The visualizations for this project were created in AWS QuickSight and were embedded into HTML/CSS to create a webpage for this COVID-19 dashboard. The webpage for this project is hosted on a AWS static S3 bucket. Data for this project was downloaded from Our World in Data and was collected starting in February 2020 and ending in mid April 2021. The datasets were downloaded and cleaned in Excel.
